# 리밸런싱 자동 계산기

기존의 리밸런싱 계산기는 귀찮게 일일히 검색해서 넣어야 합니다. 
(매우 귀찮음)<br>

BUT!! 이 프로젝트는 증권화면 캡쳐를 통해 자동으로 주식목록을 불러옵니다! 
(딸깍)

## Setup

``` 패키지 설치!!
# npm 
npm install && npm run dev
```

``` 셋업!!
# 아래를 무작정 붙여넣으시오!!! 
cp .env.example .env 
```
